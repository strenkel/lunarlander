Privacy
Lunar Lander trackt nicht und sammelt keine personenbezogen Daten. Es werden weder Cookies noch Analyse-Tools wie Google Analytics verwendet. Die Lunar Lander Web-App loggt anonymisiert jeden Seitenaufruf. Anonymisiert heißt, dass die anfragende IP-Adresse nicht mitgeloggt wird. Die Web-App ist bei Host Europe gehostet. Der Code steht auf GitHub zur Verfügung.
Lunar Lander does not track or collect any personal information. Neither cookies nor tools like Google Analytics are used. The Lunar Lander web app logs every page call anonymously. Anonymized means that the requesting IP address is not logged. The web app is hosted at Host Europe. Code is available on GitHub.
